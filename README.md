# 🍽️ Zomato Indian Cities Deep Dive

## 📌 Project Overview

This project presents a comprehensive analysis of restaurant trends across major Indian cities using the Zomato dataset. The objective is to compare restaurant density, cuisine diversity, ratings, pricing patterns, and online delivery services to uncover insights into India's restaurant ecosystem.

---

## 🎯 Objectives

- Analyze restaurant distribution across Indian cities.
- Compare average ratings and pricing trends.
- Identify cities with the highest cuisine diversity.
- Examine online delivery availability.
- Discover top-performing restaurant hubs.

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- DAX
- Data Cleaning
- Data Visualization

---

## 📂 Dataset

**Source:** Zomato Restaurants Dataset

The dataset contains information about:
- Restaurant Name
- City
- Cuisine Types
- Ratings
- Average Cost for Two
- Online Delivery Availability
- Restaurant Location Details

---

## 📊 Dashboard Features

### 🏙️ City-wise Analysis
- Total Restaurants by City
- Average Rating by City
- Average Cost for Two
- Online Delivery Percentage

### 🍜 Cuisine Diversity Analysis
- Most Popular Cuisines
- Number of Unique Cuisines per City
- Cuisine Distribution

### ⭐ Ratings Analysis
- Rating Distribution
- Top Rated Cities
- Customer Satisfaction Trends

### 💰 Cost Analysis
- Affordable vs Premium Dining
- Cost Comparison Across Cities

---

## 📈 Key Insights

- Delhi NCR contains the highest concentration of restaurants.
- Bangalore exhibits strong cuisine diversity.
- Online delivery adoption differs significantly among cities.
- Higher restaurant density does not always result in better ratings.
- Metropolitan cities generally have higher average dining costs.
